A Jekyll Theme.

## Installation/Setup
1. Complete the repo
2. Enter the folder
3. Build it: `jekyll build`
4. Start Jekyll server: `jekyll serve`
5. Configure: `_config.yml`
   * at least set `font_awesome` or download and add the font in `head.html`

Access via: [http://localhost:4000/](http://localhost:4000/)
